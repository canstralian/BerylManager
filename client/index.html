BerylManager

Manage your GL.iNet Beryl AX router like a pro

A modern package and plugin manager for OpenWRT routers. Browse, install, and manage packages without touching the command line.

📦 One-click Installs

Easily add packages and plugins with a single click.

📊 System Monitoring

Check CPU, memory, and network usage in real time.

🔄 Service Management

Start, stop, or restart services directly from the dashboard.

🔐 Secure Login

Session-based authentication with secure cookie storage.

🚀 Quick Start

Prerequisites:

  • Node.js ≥ 18
  • PostgreSQL (local or Neon-hosted)
  • pnpm (recommended)

Clone, install, and start developing:

git clone https://github.com/YOUR_USERNAME/berylmanager.git
cd berylmanager
pnpm install
cp .env.example .env
pnpm db:migrate
pnpm dev
